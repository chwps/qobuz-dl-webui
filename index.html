💿 Album

📊 qobuz-dl album info <ID>

Affiche les détails d'un album : titre, artiste, genres, qualité, label, date, tracklist.

🔊 qobuz-dl album bestquality <ID>

Affiche la meilleure qualité disponible pour un album.

⬇️ qobuz-dl album download <ID>

Télécharge tout l'album par son ID.

🎵 Track

📊 qobuz-dl track info <ID>

Affiche les détails d'une piste : titre, artiste, album, durée, qualité, UPC, ISRC.

⬇️ qobuz-dl track download <ID>

Télécharge une piste par son ID.

📃 Playlist

📊 qobuz-dl playlist info <URL>

Affiche les infos et la liste des pistes d'une playlist Qobuz.

⬇️ qobuz-dl playlist download <URL>

Télécharge toute la playlist. Attention : le contenu d'une playlist peut changer après son téléchargement (ajout/retrait de pistes).

🔍 Recherche

🔎 qobuz-dl search --keyword <TEXT>

Recherche sur Qobuz (albums, artistes, titres).

⚙️ Configuration

📄 qobuz-dl config show

Affiche le contenu actuel du fichier de configuration.

⬇️ qobuz-dl config download

Télécharge le fichier de configuration depuis Qobuz.

✏️ qobuz-dl config edit

Ouvre le fichier de configuration dans l'éditeur par défaut.

🔑 Qobuz App Credentials

🔗 qobuz-dl qobuzapp

Extrait automatiquement les credentials de l'app Qobuz (user token, user ID, device ID, session token).

👤 qobuz-dl qobuzapp user

Affiche les informations de l'utilisateur.

🔄 qobuz-dl qobuzapp refresh

Refresh les credentials (user token, user ID, device ID, session token).

🗑️ qobuz-dl qobuzapp clear

Efface les user token, user ID, device ID, session token.

🔄 Synchronisation

👤 qobuz-dl sync-artist <URL>

Sync la discographie complète d'un artiste. Télécharge tous les albums, EP, singles. Utilise les options de qualité/filtres.

Uniquement les albums (skip singles/EP)
Filtrage intelligent (smart)
Mode dry-run (prévisualisation)

📃 qobuz-dl sync-playlist <URL>

Sync une playlist Qobuz. Compare le contenu avec le dossier local et télécharge uniquement les nouvelles pistes.

Auto-confirmer les suppressions

💿 qobuz-dl sync-album <URL>

Sync un album avec le dossier local. Compare et met à jour uniquement les fichiers manquants ou modifiés.

🍀 Lucky Search & Download

🍀 qobuz-dl lucky <QUERY>

Recherche et télécharge le meilleur résultat correspondant. Alternative pratique pour obtenir rapidement un album/track.

Mode dry-run (prévisualisation)

🎤 Lyrics

🎤 qobuz-dl lyrics <DIRECTORY>

Sync les paroles (lyrics) pour les fichiers audio dans un répertoire. Ajoute les paroles aux tags des fichiers existants.

💾 Sync Database

💾 qobuz-dl sync-db

Sync la base de données de synchronisation. Met à jour les métadonnées locales avec les données Qobuz.
